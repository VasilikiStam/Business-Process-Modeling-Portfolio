# BPMN & DMN Process Modeling Portfolio

This repository contains a curated collection of **BPMN 2.0** and **DMN** process models created for academic and professional practice in **Business Process Analysis, Systems Modeling, and Decision Logic Design**.

The goal of this portfolio is to demonstrate:
- Workflow modeling using **BPMN**  
- Decision rules modeling using **DMN**  
- Understanding of administrative procedures  
- End-to-end business process analysis  
- Use of modeling tools (Visual Paradigm, Camunda Modeler, BPMN.io)
